# Sage CLI

> **Status: Experimental** — Not yet functional. Use `sage` directly.

The CLI will provide a global `sage` command installable via `curl | bash`.
This is planned for a future release. For now, use the setup script directly.

```bash
# Current (works today):
sage new my-app
sage install .
sage update

# Future (coming soon):
sage new my-app
sage install
sage update
```
