# sage-product

The PM & UX suite for Sage, extracted from the core catalog (ADR-7). Install it
when you do product discovery, research, PRDs, or UX assessment:

```bash
sage add xoai/sage-product          # once published
sage add ./packs/sage-product       # from a local checkout
```

**Skills:** jtbd, opportunity-map, prd, problem-solving, ux-brief, ux-design,
ux-discovery (absorbs the former ux-research; interview design in
references/user-interview.md), ux-plan-tasks, ux-review (merges the former
ux-audit + ux-evaluate + ux-heuristic-review), ux-specify, ux-writing.

**Workflows:** `/research` (interview → JTBD → opportunity map) and `/design`
(brief → spec → copy) ship here as pack-provided commands.

This directory is staged in-repo for review; it is destined for its own
repository, `github.com/xoai/sage-product`.
